# 快速启动指南

## 🎯 5 分钟上手

### 1️⃣ 获取项目

```bash
git clone https://github.com/ma2ong/ai-topic-generator.git
cd ai-topic-generator
```

### 2️⃣ 开始使用

在 Claude Code 中打开项目目录，输入：

```
开始今日选题生成
```

**就这么简单！** 系统会自动完成：
1. 采集全网热点 → `output/daily_hotspots/YYYY-MM-DD.json`
2. 生成 TOP10 选题 → `output/generated_topics/YYYY-MM-DD.json`
3. 审核选题质量 → `output/review_reports/YYYY-MM-DD.json`
4. 自动迭代优化直到全部通过

## 📊 输出示例

### 最终选题格式

每个选题包含：
- ✅ **事件描述**：what, when, who, background
- ✅ **核心角度**：独特视角、洞察点
- ✅ **标题建议**：主标题 + 备选标题
- ✅ **内容大纲**：章节结构、关键点
- ✅ **元数据**：难度、优先级、预估时间

### 审核报告格式

- ✅ 每个选题的详细评分（5个维度）
- ✅ PASS/REVISE/REJECT 判定
- ✅ 具体、可执行的修改建议

## ❓ 常见问题

### Q1: 热点采集失败？
- 检查网络连接
- 某些平台可能需要科学上网
- 可以指定特定平台：`只从GitHub和ProductHunt采集`

### Q2: 所有选题审核不通过？
- 热点数据质量可能不高
- 尝试放宽审核标准
- 手动介入调整部分选题

### Q3: 迭代超过 3 轮仍未通过？
- 人工介入调整
- 接受当前通过的选题
- 放弃未通过的部分
